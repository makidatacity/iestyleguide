Implementation Rules

Use these semantic tokens exactly. Build all screens with 50px controls, rounded cards, navy-based neutrals, pink primary actions, cyan informational accents, and state colors shown below. This is the closest match to Smart Lists and the current Industry Engine product UI.

Typography

Use one UI font and one numbers font throughout the app.

Role Spec Use
UI font Poppins 400 / 500 / 700 All labels, paragraphs, nav, buttons
Numbers font Rubik-Numbers 400 / 500 KPIs, scores, pagination, monetary values
Body 16px (or 14px dense tables) Main content text
Control text 14px / 700 Buttons, tabs, toolbar controls
Tiny labels 11-12px uppercase tracking Table headers, chips, metadata

Behavior and Interaction Spec

Use these values globally to match product feel.

Property Value Use
Control height 50px Buttons, search, dropdown triggers, segmented toggles
Radii 8 / 12 / 16 / pill Inputs/buttons (8), nav/table chips (12), cards (16), pills (9999)
Focus ring 2px solid border-focus All interactive elements
Hover transition 150-200ms ease Buttons, nav items, rows, tabs
Spacing system 4, 8, 12, 16, 24, 32 8px-based rhythm
Loading gradient #02b1d0 -> #8e7cff -> #d10956 Progress bars and branded loading affordances
