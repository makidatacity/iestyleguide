Implementation Rules

Use these semantic tokens exactly. Build all screens with 36px controls, 12px radius interactive elements, navy-based neutrals, pink primary actions, cyan informational accents, and state colors shown below. This is the closest match to Smart Lists and the current Industry Engine product UI.

Behavior and Interaction Spec

Use these values globally to match product feel.

Property Value Use
Control height 36px Buttons, search, dropdown triggers, segmented toggles
Radii 8 / 12 / 16 / pill Score badges (8), buttons/inputs/tabs (12), cards (16), classification pills (9999)
Focus ring box-shadow: 0 0 0 2px border-focus All interactive elements
Hover transition 150-200ms ease Buttons, nav items, rows, tabs
Spacing system 4, 8, 12, 16, 24, 32 8px-based rhythm
Loading gradient #02b1d0 -> #8e7cff -> #d10956 Progress bars and branded loading affordances
